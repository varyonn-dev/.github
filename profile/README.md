# Varyonn

## Who we are

Varyonn is a Brazilian technology and innovation company that helps organizations accelerate digital transformation through data, automation, and human-centered software engineering. We combine strategic consulting, forward-looking product delivery, and scalable platforms to turn ideas into value.

## What we do

- Data strategy, analytics, and AI/ML enablement
- Web and mobile product development (React, Node.js, cloud-native)
- Cloud architecture, infrastructure-as-code, and DevOps
- Process automation (RPA, integrations, APIs)
- Agile delivery, design thinking, and user experience

## Our values

- Customer success first
- Transparency and trust
- Continuous learning and innovation
- Empathy and collaboration
- Quality by design

## Open source & community

We embrace open collaboration and contribution. This repository includes internal onboarding and community-facing docs for Varyonn projects.

### Contribution

1. Fork the repository.
2. Create a feature branch (`git checkout -b feature/name`).
3. Open a pull request with a clear description and links to any issue(s).
4. Ensure clean commits, formatting, and tests.
